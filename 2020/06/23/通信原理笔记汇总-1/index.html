2. 抑制载波双边带调幅（DSB-SC，简写DSB）

3. AM和DSB的调制与解调总结

  1. 调制

    1. 原理：常规双边带调幅（AM）则是调制信号叠加直流分量后与载波相乘，而抑制载波双边带调幅（DSB）的调制过程实际上就是调制信号与载波的相乘运算。

    2. 要求：载频$f_C$必须大于调制信号最高频率的两倍。

    3. 调制模型

  2. 解调

    1. 包络检波（非相干解调）：只有AM可以使用，DSB不能。

    2. 相干解调：将已调信号乘上一个同频同相的载波，再通过低通滤波器恢复信号

3.3 残留边带调制（VSB）

特点

  2. 小信噪比的情况——噪声幅度远大于输入信号幅度，${\rm SNR}_i \ll 1$

    1. 此时将出现门限效应
    2. 门限效应：当包络检波器的输入信噪比下降到一个特定的阈值之后，调制信号无法与噪声分开，且有用的信号“淹没”在噪声之中。此时的输出信噪比不是按比例随着输入信噪比下降，而是急剧恶化，这种现象被称为门限效应。这个阈值被称为门限值。
    3. 相干解调不存在门限效应，门限效应一定会且仅会发生在非相干解调接收机中。

  2. 窄带调频

    窄带调频与AM的频谱表达式拥有类似的形式，但是又有着原则上的差别：

    1. 频谱：和AM类似，载波+两个边带

    2. 带宽：和AM类似，为调制信号$f(t)$最高频率分量（基带信号带宽）的两倍

    3. 区别：

      1. AM中，载波与上、下边频的合成矢量同相，只发生幅度变化

      2. 窄带调频中，由于下边频为负，因而与上边频的合成适量不与再别同相，而是存在相位偏移$\Delta \Phi$。当最大相移满足窄带调频条件时，合成矢量的幅度基本不变。

  3. 窄带调相

    1. 频谱：类似于AM，载波+两个边带
    2. 带宽：类似于AM，为调制信号$f(t)$最高频率分量（基带信号带宽）的两倍

        1. 宽带调频与调幅信号的输出信噪比之比与它们的带宽比的平方成正比。这说明，对于调频系统来说，增加传输带宽可以改善抗噪声性能。
        2. 在调频指数$\beta_{\rm FM}$较高时，调频系统的输出信噪比远大于调幅系统的输出信噪比——调频系统是用传输带宽换取的信噪比。

  2. 调频的小信噪比情况：门限效应

    1. $\beta_{\rm FM}$越高，发生门限效应的门限值也越高，转折点以上的输出信噪比改善也越明显。
    2. FM系统以带宽换取输出信噪比改善并不是无止境的。带宽$\uparrow$，$\beta_{\rm FM} \uparrow$，输入噪声功率$\uparrow$。由于输入信号功率不变，所以输入信噪比$\downarrow$。当输入信噪比恶化到一定的程度时，就会发生门限效应。

  3. 相干解调的抗噪声性能：略

4.6 预加重/去加重技术

  1. 使用预加重/去加重的原因

    1. 信号频谱大部分集中在低频段
    2. 噪声功率谱密度与调制信号的平方成正比，随着频率的增加而急剧增大，这将导致输出信噪比下降。

  2. 预加重和去加重

    1. 预加重：发送端提高输入信号的高频分量
    2. 去加重：接收端解调后压低信号高频分量

  3. 预加重和去加重网络

    1. 两各网络的关系频域上互补：$H_1 (f) H_2 (t) = 1 \quad \text{（或常数）}$

    2. 加重改善系数：用于衡量系统加重后对噪声性能的改善程度，表示为$I_{\omega}=\frac{加重后的输出信噪比{\rm SNR}_o}{不加重的输出信噪比{\rm SNR}’_o}$

      由于$So = S’_o$，因此$I{\omega} = \frac{N_o}{N’_o}$。

    3. 衰减因子：用于保持预加重后频偏不变。

      原因：提升高频分量，将使得最大频偏增大，从而超出信道所容许的频带宽度。

      方法：衰减一些信号的幅度。

    4. 网络要求

      1. 预加重：高通滤波器
      2. 去加重：低通滤波器

5.3 实际抽样

  1. 自然抽样

    抽样脉冲序列$c(t)=\sum{x=-\infty}^{\infty} p\left(t-n T{s}\right)$，其中$p(t)$是任意形状的脉冲。自然抽样时，抽样过程实际上是相乘过程。

  2. 平顶抽样

    1. 每个抽样脉冲顶部不随信号变化（自然采样顶部不平）。

    2. 平顶抽样=理想抽样+冲激响应是矩形的网络

    3. 孔径失真：抽样后信号频谱不仅扩展，而且引入了频谱失真项$\frac{\sin (\omega \tau / 2)}{\omega \tau / 2}$

      通常取$\tau=T_s$以得到最大输出信号。

      接收端必须使用频谱响应为失真项倒数，即$\frac{\omega \tau / 2}{\sin (\omega \tau / 2)}$滤波器进行频谱补偿，以抵消孔径失真。

三、量化

5.4 标量量化与矢量量化

量化——幅度离散化

  1. 标量量化

    1. 分层电平（判决阈值）：$x_k$

    2. 重建电平（量化电平）：$y_k$，数量为$L$

    3. 量化间隔：$\Deltak=k{k+1} - x_k$

    4. 量化误差：$q$

    5. 量化噪声：$\sigma_q ^2$，一般用均方误差来度量。

      $\sigma_q ^2$与量化间隔的分割有关。

    6. 最佳量化器：给定输入信号概率密度$p_x (x)$和$L$，求一组${x_k}$与${y_k}$，使得量化噪声最小。

  2. 矢量量化

    1. 输入：连续幅度的随机矢量（$N$维）：$\bar{x}$
    2. 输出：离散的幅度矢量：$\bar{y}$
      1. 重建码本（码本）：${\bar{y_i}} \ (1 \leq i \leq L)$
      2. 码本尺度：$L$
    3. 最佳量化条件
      1. 条件一：选出失真最小（距离最近）的矢量
      2. 条件二：Voroni子空间法

  3. 矢量量化比标量量化更接近率失真函数的界值$D(R)$。

5.6 均匀量化

  1. 只有在信号是均匀分布的情况下，均匀量化器才是最佳量化器。

  2. 量化噪声

    1. 量化范围：$(-V,+V)$

    2. 量化间隔：$\Delta_k = \Delta = \frac{2V}{L}$

    3. 当信号不过载时，有$\sum\limits{k=1}^{L} P{k}=1$

    4. 不过载时的量化噪声

    5. 均匀量化的不过载量化噪声功率与信号的统计特性无关。

  3. 不同信号的均匀量化信噪比

    设量化间隔数$L=2^n$，则$n = \log_2 L$。信号幅度的归一化有效值为$D=\frac{\text{信号有效值} }{最大量化电平值}$

    1. 均匀分布信号：$SNR_{\rm dB} \approx 20 \lg D + 6.02n$
    2. 正弦信号：设幅度为$A_m$，则功率为$S = A_m^2 / 2$
      1. y一般形式：$SNR_{\rm dB} \approx 4.77 + 20 \lg D + 6.02n$
      2. 当$Am = V$时，信号幅度的归一化有效值为$D = \frac{1}{\sqrt{2} }$，则满载正弦波的信噪比为：$SNR{\rm dB} = 6.02n + 1.76$
    3. 实际语音信号
      1. 动态范围：满足$SNR \geq SNR_{\rm min}$的信号功率范围。
      2. 小过载：$SNR_{\rm dB} \approx 4.77 + 20 \lg D + 6.02n$
      3. 大过载：$SNR_{\rm dB} \approx \frac{6.1}{D}$
    4. 总结
      1. 量化信噪比随量化电平数$L$的增加而提高。
      2. 每增加一位编码，量化层数$L$翻倍，量化信噪比提高6dB。

11.2 线性分组码

  1. 基本概念

    1. 奇偶校验码是最简单的线性码。其中只有一位监督码元，只能表示有错和无错。
    2. 线性码组中只存在一种运算，即模2和。（有关各位之间的运算，“+”号即表示模2和）
    3. 线性分组码：将信息码分组，每组增加若干监督位，且信息位和监督位间的关系可以用线性方程组来表示编码。
    4. 线性分组码的表示：码组长度为$n$，有$k$位信息码元和$r=n-k$位监督码元，记作$(n,k)$码。
    5. 校正子（或称伴随式）：用于检错或纠错的计算式得到的结果称为校正子，该计算式称为监督方程式。
    6. 一般来说，由$r$个监督方程式计算得到的校正子有$r$位，可以得到$2^r -1$种误码图样。对于一位误码来说，就可以用来指示$2^r -1$个误码位置。
    7. 对于$(n,k)$分组码，如果满足$2^r -1 \geq n$就可以构造出纠正一位甚至更多位的分组码。（解释：即能指示的误码位数多余实际的编码长度）
    8. 纠错码构造原理——以$(7,4)$码为例。
      1. 计算出信息码元数和监督码元数分别为$4$和$3$。
      2. 确定监督码元和信息码元的位置，分别为监督码$a_2$ ~ $a_0$，信息码$a_6$ ~ $a_3$
      3. 根据$r=3$，说明需要三个校正子，设为$S_1$、$S_2$、$S_3$。
      4. 对照校正子与误码位置的对应关系表，列写监督方程式。（看当误码位置在哪里，$S_i$校正子为1，则该位置的码元就算一个加数写在$S_i$方程的另一边，注意表示的是模2和）
      5. 改写方程组，将$S_i$都写成0，并把各方程中的码$a_2$ ~ $a_0$移到等号的另一边。注意，在模2和的计算条件下，“-”等同于“+”。
      6. 根据第5步所得到的方程组，可以得到$2^k=16$个许用码组。
      7. 当传输时接收到一个码组$a_6$ ~ $a_0$，可以根据第4步中的方程式计算出$S_1 S_2 S_3$，再对照校正子和误码位置的对照表可以判断出哪一位发生了错误。
      8. $(7,4)$码的最小码距是$d_{\min}=3$，能纠正1位误码和检测两位误码。
    9. $(n,k)$码的编码效率（编码速率）：$\frac{k}{n}=1-\frac{r}{n}$
